# How to run

Клонируйте репозиторий

```bash
git clone https://github.com/ja1zhf/banki-shop.git
```

Скачайте не достающие пакеты:

- npm:

```bash
npm install
```

- yarn:

```bash
yarn
```

Сгенерируйте билд с помощью команды

- npm:

```bash
npm run build-only
```

- yarn:

```bash
yarn build-only
```

Запустите билд с помощью команды

- npm:

```bash
npm run preview
```

- yarn:

```bash
yarn preview
```

Готово!
